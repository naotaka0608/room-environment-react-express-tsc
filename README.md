# MERNスタック・TypeScript

## PC環境
node v18.16.1

npm v9.7.2

tsc v5.1.3


## 下記コマンドで作ったものを組み合わせた
```bash
$ npx create-react-app example-react --template typescript
$ npx express example-express
```

## 実行

フロントエンド
```bash
$ cd fronend
$ npm install
$ npm start
```

バックエンド
```bash
$ cd backend
$ npm intstall
$ npm start
```


[http://localhost:3000](http://localhost:3000) で開く
